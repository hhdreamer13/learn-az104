Microsoft Entra ID — the fundamentals

Microsoft Entra ID is Microsoft's cloud identity and access service — the directory that signs users in to Microsoft 365, Azure, and SaaS apps. This file is about what it is: how it differs from on-premises Active Directory, the related Domain Services offering, and the license tiers. Managing users and groups inside it is a separate file.
