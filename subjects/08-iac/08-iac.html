Infrastructure as Code

Describe your Azure infrastructure as files you version, review, and deploy repeatably. Azure's native options are ARM templates (JSON) and Bicep (a cleaner language that compiles to ARM). Both are declarative and idempotent — deploy the same file any number of times and land in the same state.
